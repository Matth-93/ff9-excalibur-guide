⚔️ FFIX Excalibur II & Perfect Game Guide
An interactive HTML reimagining of Atomos199's legendary Final Fantasy IX walkthrough — the definitive guide for obtaining the Excalibur II while completing a perfect game.
Built with love for the FFIX community. Original guide by Atomos199, reproduced with his kind permission.

"I gladly bestow my blessings" — Atomos199

✨ Features

Tabbed navigation per disc
Collapsible segments with progress checkboxes
Treasure checklists, battle boxes, shop tables
Target time, Gil, and encounter tracking per segment
Full table of contents with deep linking
Reading progress bar

📸 Contributing Screenshots
Help illustrate every segment of the run!

Screenshots should be named after their segment ID: D1-01.png, D2-05.png etc.
Place them in the screenshots/ folder
Open a Pull Request with a brief description of what the screenshot shows
PNG or JPG accepted, ideally 16:9 or 4:3 game resolution
